👤 Identity Stable

Basic ID — The UAS ID identifies your drone to receivers. Use a serial number (ANSI/CTA-2063) or your CAA registration number, max 20 characters. The UA Type must match your aircraft. Wiki →
Max 20 chars. Serial number (recommended) or CAA registration.
⚠ Select a valid UA type — "None" will be ignored by receivers

📡 Transmission Stable

Tx Modes — Select which radio technologies to broadcast on. WiFi Beacon is the most compatible. BLE 5.0 Coded PHY offers longer range. Rates are in Hz (packets per second). Wiki →
WiFi NAN and BLE 5.0 LR are experimental. BLE 5.0 requires ESP32-S3 or ESP32-C3.

📶 WiFi AP & Serial Stable

Access Point — Configure the ESP32's management WiFi network. After connecting to ESP-RID, open http://192.168.4.1 in your browser. Set the UART baud rate to match your flight controller. Wiki →
Network name for the configuration access point.
Must match your flight controller serial port speed.
Set to Auto unless the detection fails. Manual override forces a specific protocol.
0 = auto-detect
⚠ Required when Protocol is MAVLink

🔒 System Warning

Security & Options — Lock level 0 is normal. Level 1 restricts to signed commands. Level 2 burns efuses permanently. Public keys enable signed command authentication. Wiki →
⚠ Level 2 burns eFuses permanently. This cannot be undone!
